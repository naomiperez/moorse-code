# moorse-code
## Description
An NPX/NPM Module that allows you to encode and decode "Moorse code", a cow moorse code. <br>
moo moo moo moo ♥ moo mooo ♥ moo moo moo mooo ♥ moo ♥   moo moo mooo moo ♥ moo moo mooo ♥ mooo moo ♥
